# 基于Django的Python图书借阅管理系统 python131

## 项目概述
本项目是基于Python语言和Django框架开发的图书借阅管理系统，旨在为学校或图书馆提供便捷的图书管理解决方案。

## 技术栈
- **总体架构**: Python + Django + Mysql
- **前端**: HTML + Bootstrap5 + jQuery
- **后端**: Django

## 项目环境
- **Python版本**: 3.11.4（建议3.9及以上）
- **Django版本**: 4.2.4（建议Django3及以上）
- **开发软件**: Pycharm，数据库管理可使用Navicat或Dbeaver
- **数据库**: Mysql8.2（8.0及以上）

## 功能模块
### 学生端功能
- 图书检索/查询（通过ISBN/书名/作者）
- 图书列表
- 图书借阅
- 我的借阅（归还图书/续借图书）
- 个人图书推荐（基于借阅历史）
- 统计信息（数据可视化）
- 关于
- 登录注册注销

### 管理页功能
- 图书管理
- 借阅管理
- 读者管理
- 管理员管理

## 系统角色
- 学生用户：进行图书检索、借阅、归还等操作
- 管理员：管理图书、借阅信息、用户以及系统设置

## 部署步骤
1. 环境配置：安装Python、Mysql、Django等依赖
2. 克隆项目代码到本地
3. 导入数据库文件至Mysql
4. 运行`python manage.py runserver`启动开发服务器
5. 访问学生端和管理端进行使用

## 核心亮点
- 基于Django强大的后端支持，易于维护和扩展
- Bootstrap5前端框架，界面响应式设计，适应多种设备
- 完善的权限管理，区分学生与管理员角色
- 数据可视化展示，直观了解借阅统计信息
